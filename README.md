# Final-project

## Synopsis

文字探勘期末報告，使用程式語言為 python 3 ， 系統為 windows

## Motivation

Text mining final project

## Installation

須事先準備以下 module

* BeautifulSoup4
* html5lib
* requests
* nltk
* WordCloud
* matplotlib
* scipy
* numpy
* scikit.learn


## Tests

先準備 pn.csv ，需按照附檔格式，內須要有一串專利號碼的數字

* RUN uspto.py

產生 result.csv 還有一連串的摘要文件檔

* RUN usptowordcloud.py

產生 count.csv 和文字雲

* RUN sknews .py

比較 test data 和 train data

* RUN ipc,cpc_count.py

算出 IPC, CPC 分類統計次數
